# xocolatl

A RISC-V Assembler and Interpreter.

## Get started

```bash
$ git clone "github.com/exprpapi/xocolatl"
$ cd xocolatl
$ make serve
```

## Dependencies
- `rustup`
- `npm` or `pnpm`
- `wasm-pack`
